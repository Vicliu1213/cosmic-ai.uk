🧮 數學原理
══════════════════════════════════════════════════════════════════════════════

Grover 算法步驟:

1. 初始化
   |ψ₀⟩ = (1/√N) Σ|i⟩    (均勻疊加所有態)

2. 迭代 k = π/(4√(N/m)) 次
   a. Oracle: 標記目標態的相位 (-1)
      O|i⟩ = -|i⟩  (if i ∈ marked)
   
   b. 擴散: 增幅高概率態
      D = 2|ψ₀⟩⟨ψ₀| - I
      效果: αᵢ → 2·mean - αᵢ

3. 測量
   得到目標態,成功概率 ≈ m/N

複雜度: k·(Oracle + Diffusion) ≈ O(√N)

      └─ 優化: 選擇最優信號

輸出: 
      ├─ 選中信號 ID
      ├─ 評分
      ├─ 置信度
      └─ 計算時間
