# Comic AI CLI 使用指南

## 🚀 快速开始

```bash

python src/cli/cli.py
```

打开后会立即显示**完整的菜单界面**，包含**三栏任务追踪面板**！

### 全屏显示
- ✅ 打开即显示完整界面
- ✅ 所有内容一屏显示（宽度90字符）
- ✅ 无需向上拖拽查看内容

### 三栏任务面板 (NEW!)
```
┌────────────────────────────┬────────────────────────────┬────────────────────────────┐
│🔄 即时任务 (2)                  │⏳ 未完成 (3)                   │✅ 已完成 (2)                   │
├────────────────────────────┼────────────────────────────┼────────────────────────────┤
│🟡 分析量子数据                0s  │🟡 编译代码                   -  │🟡 初始化系统                 0s  │
│🟡 优化算法参数                0s  │🟡 运行测试                   -  │🟡 加载配置                  0s  │
│                            │🟡 生成报告                   -  │                            │
│                            │                            │                            │
└────────────────────────────┴────────────────────────────┴────────────────────────────┘
```

#### 三栏说明
- **左栏 (🔄 即时任务)** - 正在进行中的任务
- **中栏 (⏳ 未完成任务)** - 待处理的任务  
- **右栏 (✅ 已完成任务)** - 已完成的任务

#### 任务优先级
- 🔴 高优先级
- 🟡 中优先级
- 🟢 低优先级

#### 任务追踪
- 显示每个任务的运行/待机时间
- 自动计算任务耗时
- 最多显示8个任务/列

### 清屏处理
- ✅ 每次操作前自动清屏
- ✅ 避免内容堆积混乱
- ✅ 整洁清爽的界面

### 美化设计
- ✅ 带边框的标题和分隔符
- ✅ 双栏菜单布局

## 🧬 物理理论

系统支持四大量子物理理论：

1. **Heisenberg** - 量子精密测量的极限
2. **Bekenstein** - 黑洞的信息内容
3. **Bremermann** - 物理计算的速度极限
4. **Landauer** - 信息处理的能源成本

## 🔧 参数修改

在参数修改菜单中可以调整：

```
• Heisenberg 精密测量精度: 10^-6 m
• Bekenstein 信息密度: 10^8 bit/m³
• Bremermann 计算速率: 10^8 op/s
• Landauer 能源效率: 10^-7 J/bit
```

## 🐛 故障排除

### 问题：界面显示不完整
**解决**：确保终端宽度至少 90 字符
```bash
# 检查终端大小
stty size
```

### 问题：任务面板不显示
**解决**：确保 tri_column_panel.py 在正确位置
```bash
ls src/core/tri_column_panel.py
```

### 问题：菜单响应缓慢
**解决**：清屏后重新启动
```bash
clear
python src/cli/cli.py
```
