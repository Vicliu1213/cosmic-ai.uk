## 故障排除指南

### 問題 1: 量子位元故障

**症狀**:
- 某個量子位元無響應
- 操作成功率下降
- 相干性喪失

**診斷步驟**:
```python
# 檢查所有量子位元狀態
qubit_status = uqg_service.check_all_qubit_health()

failed_qubits = [q for q in qubit_status if not q['healthy']]
print(f"故障量子位元: {len(failed_qubits)}")

for qubit in failed_qubits:
    print(f"  位元 {qubit['id']}: {qubit['error_reason']}")
```

**解決方案**:
```python
# 分配備用量子位元
for failed_qubit in failed_qubits:
    backup = uqg_service.allocate_backup_qubit(
        failed_qubit['system_type'],
        failed_qubit['required_qubits']
    )
    
    if backup:
        uqg_service.replace_qubit(failed_qubit, backup)
        print(f"位元 {failed_qubit['id']} 已替換")
```

### 問題 2: 操作失敗率增加

**症狀**:
- 成功率 < 99%
- 某些操作類型更容易失敗
- 系統性能波動

**診斷**:
```python
# 分析操作失敗
failure_analysis = uqg_service.analyze_operation_failures()

for op_type, stats in failure_analysis.items():
    success_rate = stats['success_count'] / stats['total_count']
    print(f"{op_type}: 成功率 {success_rate:.2%}")
    
    if success_rate < 0.99:
        print(f"  故障: {stats['failures']}")
```

**解決方案**:
```python
# 重新初始化量子位元
for system_type in ['QFT', 'ES', 'IP']:
    uqg_service.reinitialize_qubits(system_type)
    uqg_service.recalibrate_gates(system_type)

# 驗證恢復
uqg_service.verify_operation_recovery()
```
