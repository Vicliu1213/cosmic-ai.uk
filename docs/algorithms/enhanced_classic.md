# enhanced_classic 規格

`src/algorithms/enhanced_classic/` 提供一套更好的模組索引與推薦層。

## 特點
- 直接讀取 `hermes/dashboard/module_catalog.json`
- 以四層語義規則做模組推薦
- 可輸出完整 index，供上層管線或 UI 使用

## 四層規則
- energy: resource / energy / scheduling / balancing / monitoring / planning / simulation
- compression: compression / entropy / recursion / projection / reconstruction / encoding
- precision: validation / stabilization / audit / coherence / checks / sensing / quality
- compute: execution / training / routing / locality / simulation / scheduler / balancing

## 優先模組
- energy: `platform_heterogeneous`, `cosmic_engineering`, `temporal_dominance`
- compression: `cosmic_intelligence`, `quantum_holography`, `fractal_recursion`
- precision: `reality_programming`, `perfect_fortress`, `consciousness_field`
- compute: `reality_programming`, `platform_heterogeneous`, `neuro_quantum_synergy`
