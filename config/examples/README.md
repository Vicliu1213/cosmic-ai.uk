# 配置示例

各种配置场景的完整示例。

## 示例列表

- `complete/config.yaml` - 完整配置示例，展示所有可用选项
- `custom/config.yaml` - 自定义配置示例，常见的定制场景
- `migration/MIGRATION_GUIDE.md` - 从旧系统迁移配置

## 示例使用

```bash
# 查看完整示例
cat config/examples/complete/config.yaml

# 查看自定义示例
cat config/examples/custom/config.yaml

# 查看迁移指南
cat config/examples/migration/MIGRATION_GUIDE.md
```

## 最佳实践

1. 从模板开始
2. 参考示例文件
3. 根据需要调整
4. 验证配置
5. 保存备份
