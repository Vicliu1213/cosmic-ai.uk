# Comic AI CLI 使用指南

## 🚀 快速开始

```bash
# 运行全屏 CLI 界面
python src/cli/cli.py
```

打开后会立即显示**完整的菜单界面**，无需滚动！

## ✨ 新增功能特性

### 全屏显示
- ✅ 打开即显示完整界面
- ✅ 所有内容一屏显示（宽度100字符）
- ✅ 无需向上拖拽查看内容

### 清屏处理
- ✅ 每次操作前自动清屏
- ✅ 避免内容堆积混乱
- ✅ 整洁的菜单排列

### 美化设计
- ✅ 带边框的标题和分隔符
- ✅ 双栏菜单布局
- ✅ 状态信息实时显示
- ✅ Emoji 图标直观表示

## 📋 菜单选项

```
[1] 🧊  执行 Stage1 量子优势分析
[2] 📊  查看可用的物理理论
[3] ⚙️   修改分析参数
[4] 📖  查看使用说明
[5] 🚀  启动系统
[q] 🚪  离开系统
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `1-5` | 选择菜单项 |
| `q` | 退出程序 |
| `Ctrl+C` | 中断并退出 |

## 🧬 物理理论

系统支持四大量子物理理论：

1. **Heisenberg** - 量子精密测量的极限
2. **Bekenstein** - 黑洞的信息内容
3. **Bremermann** - 物理计算的速度极限
4. **Landauer** - 信息处理的能源成本

## 🔧 参数修改

在参数修改菜单中可以调整：

```
• Heisenberg 精密测量精度: 10^-6 m
• Bekenstein 信息密度: 10^8 bit/m³
• Bremermann 计算速率: 10^8 op/s
• Landauer 能源效率: 10^-7 J/bit
```

## 🐛 故障排除

### 问题：界面显示不完整
**解决**：确保终端宽度至少 100 字符
```bash
# 检查终端大小
stty size
```

### 问题：菜单响应缓慢
**解决**：清屏后重新启动
```bash
clear
python src/cli/cli.py
```
