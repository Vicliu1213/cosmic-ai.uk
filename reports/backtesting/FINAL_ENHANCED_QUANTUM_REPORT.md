### 量子操作详解

#### 1️⃣ 量子特征映射 (Quantum Feature Map)

**目的**: 将6个策略的性能指标编码成量子态

**实现**:
```
对每个特征 xi (Sharpe、收益、回撤):
  1. 参数化旋转: RY(2*xi) - 将特征映射到Y轴旋转
  2. 纠缠层: CZ门 - 策略间的相关性编码
  3. 重复2次 - 加深特征编码

结果: 创建初始纠缠态 |ψ_encoded⟩
纠缠熵从0增加到3.00 bits (满熵)
```

#### 2️⃣ 量子退火 (Quantum Annealing)

**目的**: 将系统能量最小化到目标函数的全局最优

**实现**:
```
成本哈密顿量: H_cost = -[Sharpe + Return - Drawdown]
初始哈密顿量: H_init = -叠加态的平均能量

对每个退火步骤 s = 0...1:
  H(s) = (1-s)·H_init + s·H_cost  // 从初始→目标
  
  时间演化: exp(-iH(s)·Δt) |ψ⟩
  
  结果: 系统逐步冷却到目标态

优点: 避免局部极值，找到全局最优
```

#### 3️⃣ Grover搜索 (Grover's Algorithm)

**目的**: 在量子态中快速定位最优解

**实现**:
```
Oracle掩码: 标记满足条件的配置
           score_i > median(scores)
           
Grover操作符: G = (2|s⟩⟨s| - I) · Oracle
             其中 |s⟩ 是叠加态

迭代: √N 次 (N=8个基态, 迭代≈1-2次)

效果: 将好解的振幅放大
     坏解的振幅被抵消 (干涉)
```

#### 4️⃣ 量子测量 (Measurement & Collapse)

**目的**: 获得经典结果

**实现**:
```
量子态: |ψ⟩ = Σ αi|i⟩

测量概率: P(i) = |αi|²

执行: 1024次独立测量

结果: 统计分布 → 权重向量
     频率最高的配置权重最大
```

#### 5️⃣ 经典精化 (Classical Refinement)

**目的**: 将量子建议优化到最优解

**实现**:
```
SLSQP算法:
  目标: minimize -[Sharpe + Return - Drawdown]
  初值: 量子测量的权重
  约束: Σw = 1, 0 ≤ w ≤ 0.6
  
结果: 从1.805→2.722 (+50.79%)
```

## 💡 关键洞察

### 1. 量子-经典混合的必要性

**为什么混合是最优的?**
- 纯量子: 受NISQ(近期量子)限制，精度不够
- 纯经典: 容易陷入局部最优
- **混合**: 量子全局搜索 + 经典精确优化

### 2. 策略多样化的价值

**单一策略 (100% AS)**:
- 收益最高: 216.97%
- 但回撤高: 40.45%
- Sharpe: 1.41

**多样化组合 (60% AS + 40% PMM)**:
- 收益: 172.91% (损失20.3%)
- 但回撤低: 35.70% (降低11.8%)
- **风险调整收益**: 更好

### 3. 量子纠缠的作用

**纠缠熵变化**:
- 初始化: 3.00 bits (完全纠缠)
- 测量后: 2.33 bits (部分保留)
- **效果**: 捕捉策略间的隐藏相关性

**相关性发现**:
- AS和PMM都是做市策略
- 量子干涉自动识别这种相似性
- 推荐60%-40%的互补配置
