### 為什麼實現不了200%+？

#### 1️⃣ **位置規模過小（Position Sizing Issue）**
```
當前配置：
- 初始資本：$100,000
- 每日交易：~2,000,000筆
- 每筆交易位置 = $100,000 / 2,000,000 = $0.05

結果：
- 即使贏率99.99%，極小位置規模限制收益
- 每筆交易獲利 = $0.05 × 0.03% = $0.000015
- 無法達到指數級增長
```

#### 2️⃣ **沒有使用槓桿（No Leverage）**
```
文件3聲稱的配置：
- 基礎策略：56% 年化 ✅
- 10x槓桿：56% × 10 = 560% ⚠️
- 100x槓桿：56% × 100 = 5,600% ✅（接近目標）

當前：零槓桿配置
```

#### 3️⃣ **交易成本累積**
```
成本結構：
- 做市商費用：0.01%
- 滑點：0.05%
- 總計：0.06% 每筆交易

每日成本：
= $100,000 × 0.06% × 2,000,000筆 / 252天
= 每日 ~47% 的初始資本（極高！）

實際情況：已應用費用折扣到每筆交易利潤
```

#### 4️⃣ **復利效應不足**
```
當前方法：每日固定利潤積累
- 日均利潤：$153.41
- 180天累計：$153.41 × 180 = $27,612

如果複利處理（本應如此）：
- P(n) = P₀ × (1 + r)^n
- 需要日回報率：(1.003)^252 ≈ 2.1倍 年化
- 當前日回報：0.154% < 需要的 0.3%
```

#### 5️⃣ **策略性能假設可能過樂觀**

| 策略 | 聲稱贏率 | 聲稱日均交易 | 實現利潤 |

**觀察**：低頻策略(QGR/SRB)實現最大利潤！
- 這表明：高頻交易的聲稱表現不可信
- 可能需要重新配置權重
