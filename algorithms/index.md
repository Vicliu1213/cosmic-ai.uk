# algorithms/index

這個目錄是從現有 `src/*` 與 `hermes/dashboard/module_catalog.json` 彙整出的可復用算法索引。

## 目的
- 將已存在的模組能力整理成可查找清單
- 對應到四層系統：能源、壓縮、精度、計算
- 方便後續把現有演算法接入 `src/layers/`

## 四層映射規則
- 能源層：resource / energy / scheduling / balancing / monitoring
- 壓縮層：compression / recursion / projection / entropy / codecs
- 精度層：validation / stabilization / audit / reconstruction / coherence
- 計算層：execution / training / simulation / routing / locality
