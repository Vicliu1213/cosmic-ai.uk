# 混沌共振 (Chaos Resonance)

## 概述
利用混沌系統對擾動的敏感性增強信號檢測。在市場中放大微弱趨勢信號。

## 數學基礎

### 洛倫茲方程
dx/dt = sigma(y-x), dy/dt = x(rho-z)-y, dz/dt = xy-beta*z
參數：sigma=10, rho=28, beta=8/3

### 李雅普諾夫指數
- lambda = lim(1/t)*ln|delta_x(t)/delta_x(0)|
- lambda > 0：混沌系統
- lambda < 0：穩定系統

### 隨機共振
- 信噪比改善：100-1000 倍
- 最優噪聲強度：f(信號振幅, 系統參數)

## 實現機制
