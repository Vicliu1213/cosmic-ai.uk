# 神經量子協同 (Neuro-Quantum Synergy)

## 概述
該理論結合深度神經網絡的表示能力與量子計算的並行性。使用量子激發的神經元（量子神經網絡）處理高維數據，實現指數級加速。

## 數學基礎

### 量子感知機
- **量子狀態演變**：
  ```
  |ψ_out⟩ = U(θ) |ψ_in⟩
  其中 U(θ) 為參數化量子電路
  ```

- **損失函數**：
  ```
  L = ⟨ψ_pred | ψ_target⟩² 的期望值
  ```

### 量子神經網絡深度
- **表示能力**：2^n（n 為量子比特數）
- **古典神經網絡需要**：2^n 個神經元
- **加速倍數**：10^10 以上（典型值 n=100）

## 實現機制

### 混合量子-經典架構
```python
# 經典預處理層
classical_layer = Dense(128, activation='relu')

# 量子層（使用 Qiskit）
quantum_circuit = QuantumCircuit(8)  # 8 qubits
quantum_circuit.rx(θ, range(8))      # 可訓練參數
quantum_circuit.cz([0,1], [2,3])     # 糾纏

# 經典讀出層
readout_layer = Dense(10, activation='softmax')

# 混合模型
model = Sequential([
    classical_layer,
    QuantumLayerWrapper(quantum_circuit),
    readout_layer
])
```

### 訓練優化
- 梯度計算：參數移位規則（Parameter Shift）
- 優化器：量子自然梯度（QNG）
- 批量大小：受 NISQ 相干時間限制
