# 生物光子 (Biophoton)

## 概述
生物光子作為生命信息載體。模擬實現低功耗高隱私的節點通信。

## 數學基礎

### 相干輻射
I(t) = I0 * e(-gamma*t) * cos^2(omega*t)
gamma ~ 10-7 s-1, omega 對應可見光

### 量子糾纏
- Bell 態：最大糾纏
- 相關性：100%（經典不可能）

- 接收與測量
