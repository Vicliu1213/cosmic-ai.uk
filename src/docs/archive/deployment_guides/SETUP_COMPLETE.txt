🔄 會話管理
═══════════════════════════════════════════════════════════════════════════════

啟動系統:
  cd /root/comic_ai && ./start_persistent_system.sh

後台運行:
  Ctrl+b d (在會話中按此組合鍵)

重新附加:
  tmux attach -t comic_ai_persistent

完全停止:
  tmux kill-session -t comic_ai_persistent

重新啟動:
  ./start_persistent_system.sh

🛠️ 故障排除
═══════════════════════════════════════════════════════════════════════════════

問題: 系統不啟動
解決: chmod +x /root/comic_ai/start_persistent_system.sh

問題: 窗口不響應
解決: Ctrl+c 停止 → 重新啟動

問題: 無法上傳文件
解決: mkdir -p /root/comic_ai/uploads/

問題: 日誌文件夾不存在
解決: mkdir -p /root/comic_ai/logs/

問題: 更多幫助
解決: 查看 PERSISTENT_SYSTEM_GUIDE.md
