# 文件上传服务器使用说明

## 🚀 快速启动

```bash
./start_server.sh
```

或者直接运行：
```bash
python3 file_server.py
```

## 📡 访问地址

服务器启动后，在浏览器中访问：
```
http://localhost:8000
```

## 📤 支持的文件类型

- **压缩文件**: .zip, .rar, .7z, .tar, .gz
- **图片文件**: .jpg, .jpeg, .png, .gif, .bmp, .svg
- **代码文件**: .py, .js, .ts, .java, .cpp, .c, .html, .css
- **文档文件**: .md, .txt, .pdf, .doc, .docx
- **配置文件**: .json, .yaml, .xml, .env, .ini
- **其他任何类型文件**

## 📁 文件存储

- 上传的文件自动保存在 `uploads/` 目录
- 可以在网页上查看已上传的文件列表
- 点击文件名可以查看或下载文件

## 🔧 使用步骤

1. 启动服务器
2. 打开浏览器访问 http://localhost:8000
3. 点击"选择文件"按钮选择要上传的文件
4. 点击"上传文件"按钮
5. 上传成功后可以在文件列表中看到
6. 告诉AI文件名，AI就可以读取和分析文件

## 🛑 停止服务器

在终端中按 `Ctrl+C` 停止服务器

## 💡 提示

- 支持同时上传多个文件
- 文件大小没有硬性限制（受系统内存影响）
- 上传后文件会永久保存在 uploads/ 目录
- 可以随时删除不需要的文件
