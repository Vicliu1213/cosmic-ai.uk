## System Overview

### 什麼是分層儀表版？

分層儀表版是一個 **進度導向的仪表板系統**，具有以下特點：

- **4 個操作級別**: 從初級到專家
- **漸進式功能解鎖**: 隨著進度提升而解鎖新功能
- **實時進度追蹤**: 可視化進度條顯示升級進度
- **級別特定菜單**: 每個級別顯示不同的功能選項
- **視覺反饋**: 升級時顯示動畫和通知

### 設計理念

```
初級 (Novice)
  ↓ 5 個操作
中級 (Intermediate)
  ↓ 7 個額外操作 (12 個總操作)
高級 (Advanced)
  ↓ 13 個額外操作 (25 個總操作)
專家 (Expert)
```

## FAQ

### 常見問題

**Q: 如何更快升級？**
A: 每次選擇菜單選項都計為一個操作。快速升級的方法是频繁使用系統的不同功能。

**Q: 能否跳過級別直接到專家？**
A: 不能。必須按順序通過所有級別。但達到閾值後會自動升級，無需手動操作。

**Q: 操作計數會重置嗎？**
A: 不會。操作計數是持久的，即使關閉程序後重新啟動也會保留。

**Q: 能否查看當前有多少個操作？**
A: 可以。進度條會顯示當前級別的進度百分比，菜單中也會顯示升級提示。

**Q: 各級別的數據顯示有什麼區別？**
A: 
- **初級**: 基本摘要
- **中級**: 添加索引和組件信息
- **高級**: 添加健康檢查和控制
- **專家**: 完整訪問所有數據和分析工具

**Q: 初級用戶看不到的功能會影響系統運行嗎？**
A: 不會。所有功能在後台運行，但初級用戶看不到或無法訪問。這是設計特性，鼓勵用戶升級。

**Q: 如何知道下一個級別有什麼新功能？**
A: 每個級別的菜單會顯示升級提示。升級後新功能會自動出現在菜單中。

**Q: 能重置進度回到初級嗎？**
A: 目前不能。如果需要重置，請刪除進度文件 (如果實現了持久化)。

## 高級主題 | Advanced Topics

### 進度文件位置

如果系統支持持久化進度 (需要 `.level_progress.json`):

```
/workspaces/cosmic-ai.uk/data/state/.level_progress.json
```

### 修改進度閾值

要自定義升級要求，編輯 `hierarchical_dashboard.py`:

```python
# 級別配置
LEVEL_THRESHOLDS = {
    'NOVICE': 0,
    'INTERMEDIATE': 5,
    'ADVANCED': 12,
    'EXPERT': 25
}
```

### 添加新功能

要為任何級別添加新功能：

1. 編輯 `hierarchical_dashboard.py`
2. 在相應級別的菜單中添加選項
3. 添加該選項的處理函數
4. 更新此文檔

## 故障排除 | Troubleshooting

### 問題: 無法啟動儀表版

**解決方案**:
```bash
# 檢查 Python 版本
python3 --version

# 確保在項目根目錄
cd /workspaces/cosmic-ai.uk

# 重新運行
python3 system/dashboard/hierarchical_dashboard.py
```

### 問題: 進度未保存

**原因**: 持久化功能可能未實現

**解決方案**: 提交功能請求或實現 `.level_progress.json` 支持

### 問題: 菜單選項無響應

**解決方案**:
1. 輸入有效的數字 (0-11)
2. 按 Enter 鍵確認
3. 等待操作完成後返回菜單
