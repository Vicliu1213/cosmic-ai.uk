# 系統整合儀表版 (Unified & Integrated Dashboards)

## 📊 概述

Cosmic AI 的儀表版系統提供了多層次的監控和交互界面，允許用戶以直觀的方式訪問系統的所有重要信息。

## 🎯 核心特性

- **多層次架構**: 支持概覽、詳細、詳盡三個層級
- **密交互式設計**: 友好的菜單系統，易於導航
- **實時狀態監控**: 即時加載所有系統組件的最新狀態
- **中英文雙語**: 完整的中文和英文支持
- **無縫集成**: 與所有系統組件（進度、記憶、導覽、恢復、量子）完全集成

- ✅ 量子系統狀態

### 2. A 層進度追蹤 [2]
查看完整的進度信息：
- 當前工作流程
- 進行中的任務
- 完成情況記錄
- 下一步計劃

### 3. B 層系統記憶 [3]
訪問系統激活紀錄：
- 系統演進歷史
- 過去的決策記錄
- 快速導航指南
- 關鍵事件時間軸

### 4. C 層導覽索引 [4]
快速定位任何資源：
- 完整文件結構
- 快速開始指南
- 常見問題解答
- 重要鏈接列表

### 5. 組件狀態 [5]
檢查所有系統組件：
- 加載狀態
- 文件大小
- 項目數量
- 文件路徑

### 6. 健康檢查 [6]
評估系統整體狀況：
- 7 項詳細檢查
- 健康評分百分比
- 通過/失敗統計
- 系統建議

### 7. 刷新 [7]
重新加載數據：
- 更新所有組件信息
- 同步最新狀態

- **功能數**: 7 個 (+2 新功能)

## 🔧 配置和自定義

### 添加快速命令別名
在 `~/.bashrc` 或 `~/.zshrc` 中添加：
```bash
alias cosmic-dash="python3 /workspaces/cosmic-ai.uk/system/dashboard/unified_dashboard.py"
alias cosmic-check="python3 /workspaces/cosmic-ai.uk/system/dashboard/unified_dashboard.py && echo '[6]' | python3 /workspaces/cosmic-ai.uk/system/dashboard/unified_dashboard.py"
```

然後執行：
```bash
source ~/.bashrc  # 或 source ~/.zshrc
cosmic-dash      # 啟動儀表版
```
